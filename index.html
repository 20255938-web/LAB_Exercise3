HTML Lists: Activity

I. Objectives

  1. Understand the use of ordered and unordered lists in HTML.
  2. Learn how to apply list attributes like type and start.
  3. Practice creating multi-level nested lists.

II. Concepts to be Applied

  • HTML list tags: <ol>, <ul>, <li>
  • Applying different numbering styles
  • Combining lists with text formatting tags
